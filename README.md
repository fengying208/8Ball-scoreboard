# 撞球計分網站 (8Ball Scoreboard)

這是一個專為平板與觸控裝置優化的網頁版撞球計分系統。

---


* **平板操作優化**：
    * **雙擊加分**：快速連續點擊選手區域即可 +1 分。
    * **長按減分**：按住選手區域 **2 秒** 即可 -1 分（具備防止誤觸邏輯）。
* **動態開球追蹤 (Break Indicator)**：
    * 支援三種主流規則：**贏家開球**、**輪流開球**、**輸家開球**。
    * 以**黃橘色發光方框**即時顯示當前開球者。
* **完整的賽制設定**：
    * 自定義選手姓名。
    * 自定義搶幾賽制（Race to）。
    * 指定首局開球者。
* **即時比賽計時**：進入比賽後自動開始計時（時:分:秒）。

---

##  操作說明

### 1. 比賽設定 (大廳)
在進入計分介面核心前，你可以在大廳設定：
* **選手姓名**：輸入兩位參賽者的名稱。
* **賽制 (搶幾)**：設定目標局數（例如：搶 10）。
* **開球規則**：選擇得分後，下一局的開球權歸屬。
* **首局開球者**：指定誰先開始第一局。

### 2. 計分板操作
| 動作 | 觸發效果 | 備註 |
| :--- | :--- | :--- |
| **快速點擊兩次 (Double Click)** | 選手分數 **+1** | 會自動觸發開球方塊切換 |
| **長按不放 2 秒 (Long Press)** | 選手分數 **-1** | 專為平板設計，防止誤觸 |
| **點擊「回大廳」** | 結束並重置比賽 | 會彈出確認視窗防止手
